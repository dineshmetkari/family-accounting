Family Accounting

Family Accounting is the application to manage family balance. Android version 1.6. It contains the functionalities as below:

1. Records Entry/Display
Provides the funtion to entry family financial information and aslo display.

2. Customizing
Provides the customizing to configure master data. For example, bank account information.

3. System Settings
Provides the settings to configure different transaction type.(Not support in current version).

4. Reporting Provides family financial reports.

Tech:

1. Storage Format
2. Loading

Used Libraries:

1. A Chart Engine, which is for reporting.
