Customizing

There is two types of customizing, Master Data Customizing & Transaction Customizing.

Master Data Customizing:

User can change the data in master data customizing.
1. G/L Account
2. G/L Account Group
3. Vendor
4. Customer
5. Business Area
6. Bank Account
7. Bank Key

System Default Customizing

1. G/L Account
2. G/L Account Group
3. Vendor
4. Customer
5. Business Area
6. Bank Account
7. Bank Key

System Customizing

User cannot change the data in system customizing.
1. G/L Account Type
2. Critical Level
3. Bank Account Type
